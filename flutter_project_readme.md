Work Nestor is a modern job platform designed to connect talented professionals with leading employers, similar to BD Jobs.
It aims to make the job search and recruitment process smarter, faster, and more efficient for both job seekers and employers.

### Features

- Smart Search Filters: Quickly find relevant jobs based on location, role, experience, and more.

- Easy Job Applications: Apply to multiple jobs seamlessly with just a few clicks.

- Interview Panel: Manage and track interviews efficiently.

- Career Advice: Get expert guidance to enhance your career growth.

- Salary Tools: Compare salaries and make informed decisions.

- Resume Tools: Build and optimize your resume for better opportunities.

### Tech Stack

- Frontend: Flutter (Mobile App) / React (Web)

- Backend: Node.js / Express

- Database: MySQL / Firebase

- Authentication: JWT / OAuth
